# ACT12_navegacion3pantallas_joeldelgado
lenguaje dart flutter nivel principiante, navegacion entre 3 paginas utilizando rutas nombradas, desde main  llamar a la pagina 1, en la pagina 1 en app bar mostrar el texto "inicio Joel Delgado 6j" en color blanco, color de fondo azul, iconos blancos, en body un contenedor redondeado color azul claro de 200x200 con texto negro y centrado y un boton de color amarillo texto negro para seleccionar pagina 2, en la pagina 2 un app bar con texto "segunda pagina 6j "en  color rojo, fondo negro y los iconos blancos, en body una imagen desde la red , y un boton para avanzar a la pagina 3, en la pagina 3 un app bar con  texto color negro "pagina 3 el joel", color de fondo verde claro, todo en un solo archivo, elegane y atractivo
